GateManager.MonitorSetting.GCCollectCount

The time internal in seconds to execute a GC Collect of GATE Engine

GateManager.MonitorSetting.Library

The communication LIBRARY used by COM (to connect to SERVER) of GATE Engine

Default is: 'standard'

GateManager.MonitorSetting.Protocol

The PROTOCOL used by COM (to connect to SERVER) of GATE Engine

Default is: 'tcp'

GateManager.MonitorSetting.ServerDN

The SERVER DN used by COM (to connect to SERVER) of GATE Engine

Default is: 'localhost'

GateManager.MonitorSetting.ServerPort

The SERVER Port used by COM (to connect to SERVER) of GATE Engine

Default is: '10000' for 'tcp' protocol of 'standard' library, or

Default is: '20000' for 'ipc' protocol of 'standard' library, or

Default is: '40000' for 'tcp' protocol of 'wcf' library, or

Default is: '45000' for 'http' protocol of 'wcf' library

Default is: '50000' for 'ws' protocol of 'streaming' library, or

Default is: '55000' for 'wss' protocol of 'streaming' library, or

Default is: '50000' for 'pipe' protocol of 'streaming' library, or

Default is: '55000' for 'spipe' protocol of 'streaming' library, or
